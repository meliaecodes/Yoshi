# Yoshi, Engagement Agent

To get started using Yoshi, Engagement Agent

1. Go to [app.relevanceai.com/](https://app.relevanceai.com/) and login
2. Ensure you're in the Agents Menu by selecting Agents in the menu to the left
3. Click on the '+ New Agent' button at the top right of the screen
4. Select 'Import' and follow the prompts to upload [Yoshi](./agent_0cc127f5-3b85-474c-912f-9c54cb0df866.rai)

Once the agent is ready, head to [YouTube](https://youtu.be/e-pp8BPnDC8) to watch the guide on how to add the YouTube Data API call Tool to this agent.
